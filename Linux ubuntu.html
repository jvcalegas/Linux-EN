Linux Ubuntu Tutorial

Remembering that we performed the installation of Ubuntu through a virtual machine (Virtual Box)

1st step- Click on “New”.

2nd step- Select the machine name, folders where the files will be saved, operating system types, version and click “Next”.

3rd step- In this step we must select the RAM memory size to be used by O.S., you must take into account the amount of memory that O.S. needs and the memory available in the system. The best choice would be to choose the recommended size. After choosing the memory size, click “Next”.

4th step- In this screen select “Create a virtual hard disk now” or select an existing file if you have one. Click “Create”.

5th step- In this screen, select “VDI” and click “Next”.

6th step- Select “Dynamically allocated” and click “Next”.

7th sO.S. and the host memory. Click on "Create".tep- In this screen, choose the disk size respecting the minimum required by the virtual

8th step- After create your virtual machine , click “Start”.

9th step- In this step you will put the optical disk file of the O.S. that you chose for your virtual machine, in this case will be Ubuntu. Click on “Start”.

10th step- Select “Install Ubuntu” or, if you prefer, select the one who will serve you better.

11th step- Now choose you keyboard layout. Click “Continue”.

12th step- Select “Normal Installation” and “Download updates while installing Ubuntu”, or if you prefer, choose the one who will serve you better. Click “Continue”.

13th step- Select “Erase Disk and install Ubuntu”, or “Something else” if will be better for you. Click “Install Now”.

14th step- Choose your location. Click “Continue”.

15th step- Now you will put your user name, the name of your computer and a password. Click “Continue”.

16th step- After the installation was done, you virtual machine will restart. Click “Restart now” and wait.

It’s done! Now, enter in your user and enjoy your virtual machine!
